Media queries

Instructions

The procedure is the following:

  • Copy index.html and styles.css from ex3 to ex5
  • Add responsiveness to styles.css with media queries
    • if size is small phone, section, aside, and nav elements reside in one column; the order is s1, a1, s2, a2, s3, n1, n2
    • if the size is phone or tablet, elements are set in two columns, the order remains the same, however, nav elements to go to a separate row
    • if >tablet, in three columns; the order changes a bit, first row consists of s1, s2, and a1, next row of s3 and a2; the last row of n1 and n2
    • if there is need for empty elements, these are put in the middle.
    • Save your responsive style declarations in styles.css.
  • Commit your changes to git repo
  • Submit your work to Plussa
